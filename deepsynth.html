The DeepSynth project

Official announcement

Principal investigator: Nathanaël Fijalkow
Duration: 2019 -- 2022 (3 years)
Budget: 180k € plus a two-year postdoc
Objectives: The conception of computer programmes is a complicated, costly, and error-prone task. Program synthesis is an ideal where the program is automatically generated from its specification. It is a very hard problem requiring heavy resources both in computational time and memory. Recently, experiments achieved impressive results, showing that it is possible to synthesise complicated programs.
The objective of the DeepSynth project is to use learning techniques for program synthesis.
